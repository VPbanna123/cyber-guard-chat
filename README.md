cyber-guard-chat

AI-powered real-time chat application that detects and prevents cyberbullying using a machine learning–based toxicity detection model.

This project is built using React + Vite for a fast development environment with HMR and ESLint support.

React Plugins Used

@vitejs/plugin-react — uses Babel for Fast Refresh

@vitejs/plugin-react-swc — uses SWC for Fast Refresh

React Compiler

The React Compiler is disabled by default for better dev and build performance.
To enable it, refer to the official documentation:
https://react.dev/learn/react-compiler/installation

ESLint Configuration

For production-level apps, TypeScript with type-aware lint rules is recommended.
Refer to the Vite React+TS Template for integration details.
